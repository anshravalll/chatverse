New Retrieval Augmented Generation (RAG) algorithm for your local ChatGpt like assistant. Queue like structure for context management for Meta Llama3 (https://llama.meta.com/). Enjoy :)

Features:

   1.  RAG Algorithm: Combines information retrieval with generation to enhance response accuracy and relevance.
  
   2.  Queue-based Context Management: Efficiently manages conversation context using a queue structure.
  
   3.  Meta Llama3 Model: Utilizes Meta's Llama3 model for generating responses.
  
   4.  PostgreSQL Integration: Stores chat history using PostgreSQL for persistent data management.
  
   5.  FAISS Vector Store: Uses FAISS for similarity search to improve context retrieval.
  
   6.  Flask API: Provides endpoints for interaction with the chat assistant.

   7.  Ollama: Ollama for fast LLM inference.
